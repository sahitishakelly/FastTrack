#  FocusTrack - Study Time Tracker (Java)

FocusTrack is a simple Java console-based application that helps users track their study tasks and time spent on each task.

## Features
- Add study tasks
- Record time spent (in minutes)
- View all tasks
- Calculate total study time

##  Technologies Used
- Java
- ArrayList (for data storage)
- Scanner (for user input)

##  How It Works
The user can choose options from a menu:
1. Add a task with time
2. View all tasks
3. Exit the application

The program stores tasks dynamically and calculates the total time spent studying.

##  Purpose
This project is built to improve time management and demonstrate basic Java programming skills, including Object-Oriented Programming concepts.

##  How to Run
1. Compile the program:
   javac FocusTrack.java

2. Run the program:
   java FocusTrack

##  Sample Output
Add Task → Enter task name and time  
View Tasks → Displays all tasks with total time
